# Use trial and sandbox demo environments

Beyond Microsoft 365 demo accounts, you have other options for demonstration environments: prospect trials and sandbox environments. This article covers both approaches.

## Add a Business Central trial to your prospect's organization

Add a Business Central trial directly to a prospect's organization so they can evaluate the product in their own environment.

### Sign up for a trial

[!INCLUDE [admin-trial-signup](../includes/admin-trial-signup.md)]

Use this type of environment to show the general user interface or how apps from Marketplace add capabilities.

### Important: license assignment for trial conversion

[!INCLUDE [admin-license-trial](../includes/admin-license-trial.md)]

### Trial duration and extension

If a prospect signs up for a free trial and uses [!INCLUDE [prod_short](../developer/includes/prod_short.md)] to help run their business, they need to decide whether to subscribe within the first 60 days. If they extend their trial once and still aren't sure, they need to contact a partner.

> [!NOTE]
> Use your own Microsoft 365 demo account to show prospects more tailored experiences. Learn more in [Set up Microsoft 365 demo environments](demo-environment-mdx.md).

## Demo in sandbox environments

Set up tailored demo environments in a sandbox environment if you already have a [!INCLUDE [prod_short](../developer/includes/prod_short.md)] tenant. This approach is ideal when you want to show different implementations or isolated scenarios.

### When to use sandbox demos

Use sandbox environments for demos when you:
- Already have a Business Central tenant
- Want to show two different implementations of a capability
- Need to illustrate an integration point
- Want to isolate demo changes from your production environment

> [!NOTE]
> Sandboxes are limited in functionality, including performance. Learn more in [Sandbox environments](environment-types.md#sandbox-environments).

### Partner sandboxes for demos

Starting in February 2022, partners can request dedicated licenses to set up more sandbox environments. Learn more in [Partner sandboxes](environment-types.md#partnersandbox).
