# Compiler Warning AL1085

[!INCLUDE[banner_preview](../includes/banner_preview.md)]

The package file {0} is locked and cannot be accessed.


## Description
The package file is locked and cannot be accessed. If this is preventing dependency resolution close any applications that may be using the file and try again.
