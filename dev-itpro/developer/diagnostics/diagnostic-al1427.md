# Compiler Designer customization warning AL1427

[!INCLUDE[banner_preview](../includes/banner_preview.md)]

You cannot set the '{0}' property to '{1}' for {2} '{3}' in the page customization '{4}'. Reason: '{5}'. Ignoring the property.


## Description
You cannot reference an element defined in a page customization outside the page customization declaring it. Ignoring the property.
