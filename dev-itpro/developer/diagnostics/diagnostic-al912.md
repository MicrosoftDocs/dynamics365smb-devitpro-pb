# Compiler Warning (future error) AL0912

[!INCLUDE[banner_preview](../includes/banner_preview.md)]

The name of a {0} cannot be empty because it can cause runtime errors.


> [!IMPORTANT]
> This warning will become an error with Business Central 2027 release wave 1.  

## Description
Fields and keys cannot have empty names as this can cause runtime errors.
