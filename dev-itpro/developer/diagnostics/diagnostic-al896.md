# Compiler Error AL0896

[!INCLUDE[banner_preview](../includes/banner_preview.md)]

The field '{0}' cannot be defined because it has a calculation formula that leads to a cycle. The field '{1}' from table '{2}' is involved in the cycle.


## Description
A flowfield's calculation formula cannot be calculated if it causes a cycle.
