# OnInitXmlPort (Xml Port) Trigger
> **Version**: _Available or changed with runtime version 1.0._

Runs when the XMLport is loaded and before any table views and filters are set.


## Syntax
```AL
trigger OnInitXmlPort()
begin
    ...
end;
```
