# SUBSTRING Method
Retrieves a substring from this instance.  
```  
SUBSTRING := Text.SUBSTRING(StartIndex, [Count])  
```  
## Parameters
*StartIndex*    
&emsp;Type: Integer  
The one-based starting character position of a substring in this instance.  
  
*Count*    
&emsp;Type: Integer  
The number of characters in the substring.  
  
## Return Value
*Substring*  
&emsp;Type: Text
