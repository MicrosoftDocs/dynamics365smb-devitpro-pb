# CAPTION Method (TestPage)
Gets the caption of the test page.  
  
## Syntax  
  
```  
Caption := TestPage.CAPTION  
```  
  
#### Parameters  
 *TestPage*  
 Type: TestPage  
  
 The test page that the caption belongs to.  
  
## Property Value/Return Value  
 Type: Text or Code  
  
 The caption of the test page.
