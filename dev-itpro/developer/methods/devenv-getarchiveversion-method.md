# GETARCHIVEVERSION Method
Returns the version of the [!INCLUDE[d365fin_md](../includes/d365fin_md.md)] extension that the specified table is part of.  
  
## Syntax  
  
```  
[Version :=] GETARCHIVEVERSION  
```
