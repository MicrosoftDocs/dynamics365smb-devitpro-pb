# ISFILE Method (Variant)
Indicates whether an AL variant contains a File variable.  
  
## Syntax  
  
```  
  
Ok := Variant.ISFILE  
```  
  
#### Parameters  
 *Variant*  
 Type: Variant  
  
## Property Value/Return Value  
 Type: Boolean  
  
 **True** if the AL variant contains a File variable; otherwise, **false**.
