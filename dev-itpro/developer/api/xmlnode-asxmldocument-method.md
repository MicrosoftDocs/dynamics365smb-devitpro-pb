# AsXmlDocument Method
Converts the node to an XmlDocument node. The operation will fail if the node is not an XmlDocument.  
```  
XmlDocument := XmlNode.AsXmlDocument()  
```  
## Return Value
*XmlDocument*  
&emsp;Type: [XmlDocument](xmldocument-class.md)
