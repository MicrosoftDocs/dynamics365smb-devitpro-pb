# Value Property
Gets or sets the value of this node.  
```  
[Value := ] XmlText.Value([NewValue])  
```  
## Parameters
*NewValue*    
&emsp;Type: [String](../datatypes/devenv-text-data-type.md)  
The new value of this node.  
  
## Property Value
*Value*  
&emsp;Type: [String](../datatypes/devenv-text-data-type.md)
