# Timeout Property
Gets or sets the duration in seconds to wait before the request times out.

```
HttpClient.Timeout
```

## Property Value
&emsp;Type: [Integer](../datatypes/devenv-integer-data-type.md)  

&emsp;The number of seconds before the request times out.
