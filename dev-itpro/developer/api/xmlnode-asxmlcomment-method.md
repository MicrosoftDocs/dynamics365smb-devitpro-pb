# AsXmlComment Method
Converts the node to an XmlComment node. The operation will fail if the node is not an XmlComment.  
```  
XmlComment := XmlNode.AsXmlComment()  
```  
## Return Value
*XmlComment*  
&emsp;Type: [XmlComment](xmlcomment-class.md)
