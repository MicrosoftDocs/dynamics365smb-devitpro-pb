# UICop Warning AW0009
Using a Blob with subtype Bitmap on a page field is deprecated. Instead use the Media/MediaSet data types.

## Description
Using a Blob with subtype Bitmap on a page field is deprecated. Instead use the Media/MediaSet data types.
