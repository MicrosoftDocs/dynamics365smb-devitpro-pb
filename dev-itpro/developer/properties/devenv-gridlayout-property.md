# GridLayout Property
The controls inside this definition should be layed out as a grid using the caption of the first group as row caption. 

## Applies To  
  
-   Group controls on pages  
  
## Property Values  
-   Rows  
-   Columns  
  
## Remarks 
This property is only supported on grids.
