# ExecutionTimeout Property

Sets the maximum time the report will run after which it is automatically terminated. The format must be a timeout duration specified as `hh:mm:ss`.
  
## Applies To  

- Reports

## Property Value   
 
## Syntax
```
ExecutionTimeout := `10:05:55`;
```
