# BlankZero Property
Indicates whether the system displays zeros (0) and No.  
  
## Applies to  
  
-   Page Fields  
  
## Property Value  
 **True** if zeros and No are not displayed; otherwise, **false**. The default is **false**.  
  
## Syntax
```AL
BlankZero = true;
```
