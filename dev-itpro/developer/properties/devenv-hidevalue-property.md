# HideValue Property
Sets whether to show or hide a value for the user based on an expression.  
  
## Applies to  
 Page Fields  
  
## Property Values  
 **True** if the value is hidden; otherwise, **false**. The default is **false**.  

## Syntax
```
HideValue = true;
```
