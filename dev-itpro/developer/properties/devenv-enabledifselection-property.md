# EnabledIfSelection Property
Sets a value that indicates whether the selected action is enabled or disabled.  
  
## Applies to  
  
-   Actions  
  
-   Pages  

## Syntax
```
EnabledIfSelection = true;
```
 
## Property Value  
 **True**/**false**
