# List.Add Method
Adds a value to the end of the List.


## Syntax
```
 List.Add(Value: T)
```
## Parameters
*List*  
&emsp;Type: [List](list-data-type.md)  
An instance of the [List](list-data-type.md) data type.  

*Value*  
&emsp;Type: [T](list-data-type.md)  
The value to be added to the end of the List.
