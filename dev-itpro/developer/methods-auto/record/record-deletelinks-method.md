# DeleteLinks Method
Deletes all of the links that have been added to a record.


## Syntax
```
 Record.DeleteLinks()
```

## Parameters
*Record*  
&emsp;Type: [Record](record-data-type.md)  
An instance of the [Record](record-data-type.md) data type.
