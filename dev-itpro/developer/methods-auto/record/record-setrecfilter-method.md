# Record.SetRecFilter Method
Sets the values in the current key of the current record as a record filter.


## Syntax
```
 Record.SetRecFilter()
```

## Parameters
*Record*  
&emsp;Type: [Record](record-data-type.md)  
An instance of the [Record](record-data-type.md) data type.
