# FieldClass Option Type
> **Version**: _Available from runtime version 1.0._

Represents the type of a field class.

## Members
|  Member  |  Description  |
|----------------|---------------|
|Normal|A normal field.|
|FlowField|A flow field.|
|FlowFilter|A flow filter.|
