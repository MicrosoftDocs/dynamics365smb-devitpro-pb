# Native attribute
> **Version**: _Available or changed with runtime version 17.0._

Specifies that the method is a native method. Restricted to Microsoft only


## Applies to

- Method

> [!NOTE]
> The **Native** attribute can only be set inside codeunits.

## Syntax

```AL
[Native]
```
