# Dynamics 365 Business Central developer and IT Pro documentation

This repo contains the source for the [Dynamics 365 Business Central developer and IT Pro documentation](https://learn.microsoft.com/dynamics365/business-central/dev-itpro/) on Microsoft Learn.

For more information about contributing to this repo, see [Contribute to the Help for Dynamics 365 Business Central](https://learn.microsoft.com/dynamics365/business-central/dev-itpro/help/contributor-guide).

## Repository structure

The `dev-itpro/` folder is the published docset on Microsoft Learn. The other folders contain archived or supporting content.

```text
dynamics365smb-devitpro/
├── dev-itpro/                        # Published docset (Microsoft Learn)
│   ├── administration/               #   Tenant management, cloud migration, telemetry
│   ├── ai/                           #   Copilot, AI agents, AI Development Toolkit
│   ├── al-go/                        #   AL-Go for GitHub DevOps automation
│   ├── api-dataverse/                #   Dataverse and Dynamics 365 integration APIs
│   ├── api-irs1099/                  #   IRS 1099 reporting API reference
│   ├── api-reference/                #   Business Central REST API reference (v1.0, v2.0)
│   ├── api-sustainability/           #   ESG and sustainability reporting APIs
│   ├── auditing/                     #   Activity tracking and compliance auditing
│   ├── compliance/                   #   AppSource validation and compliance
│   ├── cside/                        #   Legacy C/SIDE development tools
│   ├── deployment/                   #   Architecture, installation, system requirements
│   ├── developer/                    #   AL language, extensions, testing, best practices
│   ├── early-preview/                #   Early access and preview features
│   ├── embedapps/                    #   Embed App program for ISVs
│   ├── help/                         #   Help system and contribution guidelines
│   ├── includes/                     #   Reusable content snippets
│   ├── media/                        #   Screenshots, diagrams, and visual assets
│   ├── performance/                  #   Performance optimization and profiling
│   ├── powerplatform/                #   Power Automate, Power Apps, Dataverse integration
│   ├── security/                     #   Authentication, data protection, privacy
│   ├── terms/                        #   Software lifecycle and support policies
│   ├── upgrade/                      #   Version upgrade and data migration guides
│   ├── webservices/                  #   OData, SOAP, REST, and integration patterns
│   └── whatsnew/                     #   Release notes and feature announcements
│
├── media-source/                     # Source diagrams and images (pptx, vsdx, svg)
├── rest-reference-source/            # REST API reference source files
├── tools/                            # Utilities and scripts
│
├── api-archive/                      # Archived API docs
├── archive/                          # Archived deprecated feature docs
├── onprem/                           # Archived on-premises (NAV) docs
├── properties/                       # Archived property reference
├── triggers/                         # Archived trigger reference
├── troubleshooting-archive/          # Archived troubleshooting guides
└── Upgrade-archive/                  # Archived upgrade docs
```
