# LinkedObject Property
Use this property to link to SQL Server objects.  
  
## Applies To  
 Tables  
  
## Remarks  
 When you set the LinkedObject property to **Yes**, the [LinkedInTransaction Property](LinkedInTransaction-Property.md) becomes available.
